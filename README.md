# lein-project-clean

A lein plugin to tidy/refactor source files in a project via refactor-nrepl middleware

## Usage

Put `[lein-project-clean "0.1.0"]` into the `:plugins` vector of your
`:user` profile.

In a project you want to clean:

    $ lein project-clean
